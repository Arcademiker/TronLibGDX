# TronLibGDX
A Java implementation of a basic Tron game with LibGDX

The jar programm should work out of the box
```
on linux:
java -jar Tron.jar 

controlls:

   blue player: w = up
                a = left
                s = down
                d = right
               
  green player: arrow up    = up
                arrow left  = left
                arrow down  = down
                arrow right = right
  ```
